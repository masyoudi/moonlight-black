# Moonlight Black 

## How to Install

Currently is only support manual installation:


### Clone Repository
```sh
git clone git@github.com:masyoudi/moonlight-black.git
```

### Move to directory
```sh
cd moonlight-black
```

### Install vsce
```sh
npm i -g vsce
```

### Create vsix file
```sh
vsce package
```

### VS Code
- Open your VS Code
- Move to `Extensions` tab
- Open menu from top right of sidebar menu
- Click `Install from VSIX...`
